# clear_screen

**言語:** [English](../../en/actions/clear_screen.en.md) | 日本語

## 説明

画面とすべてのスクロールバックをクリアします。

## 使用方法

このアクションは引数なしで呼び出されます：

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/clear_screen.ja.md
# 画面とすべてのスクロールバックをクリア
unbind ctrl+l
keybind ctrl+shift+k = clear_screen
```
