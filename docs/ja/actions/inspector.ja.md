# inspector

**言語:** [English](../../en/actions/inspector.en.md) | 日本語

## 説明

ターミナルインスペクタの表示を制御します。

有効な引数: `toggle`, `show`, `hide`。

## 使用方法

このアクションは表示制御引数が必要です：

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/inspector.ja.md
# ターミナルインスペクタを制御
keybind f12 = inspector:toggle
```
