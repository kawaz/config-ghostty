# undo

**言語:** [English](../../en/actions/undo.en.md) | 日本語

## 説明

フォーカスされたサーフェスまたはターミナルの最後の取り消し可能なアクションを取り消します（可能な場合）。これはタブやウィンドウを閉じるなどのアクションを取り消すことができます。

Ghostty のすべてのアクションが取り消し/やり直し可能なわけではありません。現在、取り消し/やり直しをサポートするアクションのリストは以下に限定されています：

- ウィンドウの新規作成、ウィンドウを閉じる
- タブの新規作成、タブを閉じる
- スプリットの新規作成、スプリットを閉じる

すべてのアクションは限定された時間のみ取り消し/やり直し可能です。たとえば、閉じたスプリットを復元できるのは、スプリットが閉じられてから数秒間のみです。正確な期間は `undo-timeout` 設定で設定されます。

取り消し/やり直しアクションが制限されていることにより、時間の経過に伴うメモリ使用量が限定され、閉じたサーフェスが無期限にバックグラウンドで実行されることがなくなり、キーバインドがターミナルアプリケーションで使用できるようになります。

macOS のみで実装されています。

## 例

```conf
keybind = cmd+z=undo
```
