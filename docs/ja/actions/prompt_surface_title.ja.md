# prompt_surface_title

**言語:** [English](../../en/actions/prompt_surface_title.en.md) | 日本語

## 説明

ポップアッププロンプトを介して、現在フォーカスされているサーフェスのタイトルを変更します。

これは Linux で libadwaita 1.5 以降が必要です。現在の libadwaita バージョンは `ghostty +version` を実行することで確認できます。

## 使用方法

このアクションは引数なしで呼び出されます：

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/prompt_surface_title.ja.md
# サーフェスのタイトルを変更するためのプロンプト
keybind alt+shift+t = prompt_surface_title
```

## プラットフォーム

Linux のみ
