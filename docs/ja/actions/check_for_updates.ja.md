# check_for_updates

**言語:** [English](../../en/actions/check_for_updates.en.md) | 日本語

## 説明

更新を確認します。

macOS でのみ実装されています。

## 使用方法

このアクションは引数なしで呼び出されます：

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/check_for_updates.ja.md
# 利用可能な更新を確認
keybind alt+shift+u = check_for_updates
```

## プラットフォーム

macOS のみ
