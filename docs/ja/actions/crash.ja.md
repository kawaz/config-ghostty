# crash

**Language:** [English](../../en/actions/crash.en.md) | 日本語

## 説明

フォーカスされたサーフェスの目的のスレッドで Ghostty をクラッシュさせます。

**警告：** これはハードクラッシュ（パニック）で、データが失われる可能性があります。

このアクションの目的は、クラッシュ処理をテストすることです。ユーザーによっては、クラッシュ報告機能をテストして、すべてが期待どおりに機能するかどうかを判定するのに役立つ場合があります。

値はクラッシュ位置を決定します：

* `main` - メイン（GUI）スレッドでクラッシュします。
* `io` - フォーカスされたサーフェスの IO スレッドでクラッシュします。
* `render` - フォーカスされたサーフェスのレンダースレッドでクラッシュします。

## 使い方

```conf
keybind = ctrl+shift+c=crash:main
```
