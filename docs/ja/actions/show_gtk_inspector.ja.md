# show_gtk_inspector

**Language:** [English](../../en/actions/show_gtk_inspector.en.md) | 日本語

## Description

GTK インスペクターを表示します。

macOS では効果がありません。

## Example

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/show_gtk_inspector.ja.md
# GTK インスペクターを表示
keybind = super+i=show_gtk_inspector
```
