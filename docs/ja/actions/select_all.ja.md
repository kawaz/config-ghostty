# select_all

**Language:** [English](../../en/actions/select_all.en.md) | 日本語

## Description

画面上のすべてのテキストを選択します。

## Example

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/select_all.ja.md
# 画面上のすべてのテキストを選択
keybind = super+a=select_all
```
