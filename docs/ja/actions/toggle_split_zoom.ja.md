# toggle_split_zoom

**Language:** [English](../../en/actions/toggle_split_zoom.en.md) | 日本語

## 説明

現在の分割をズームイン/ズームアウトします。

分割がズームインされると、現在のタブの全スペースを占有し、他の分割が非表示になります。タブまたはタブバーもこれを反映して、ズーム状態を示すアイコンを表示します。

## 使用方法

```
toggle_split_zoom
```
