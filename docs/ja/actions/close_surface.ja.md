# close_surface

**言語:** [English](../../en/actions/close_surface.en.md) | 日本語

## 説明

現在の「サーフェス」(ウィンドウ、タブ、分割など) を閉じる。

[`confirm-close-surface`](../config/confirm-close-surface.ja.md) 設定値によって、クローズ確認ポップアップがトリガーされる可能性がある。

## 使用例

```conf
keybind = super+w:close_surface
```
