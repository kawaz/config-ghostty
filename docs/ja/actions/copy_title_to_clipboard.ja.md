# copy_title_to_clipboard

**Language:** [English](../../en/actions/copy_title_to_clipboard.en.md) | 日本語

## Description

ターミナル タイトルをクリップボードにコピーします。ターミナル タイトルが設定されていないか空の場合、このは効果がありません。

## Example

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/copy_title_to_clipboard.ja.md
# ターミナルのタイトルをクリップボードにコピー
keybind = super+c=copy_title_to_clipboard
```
