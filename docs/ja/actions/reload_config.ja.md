# reload_config

**Language:** [English](../../en/actions/reload_config.en.md) | 日本語

## 説明

設定を再度読み込みます。

正確な意味は使用されるアプリランタイムに依存していますが、これは通常、設定ファイルを再度読み込んで、変更を適用することを含みます。すべての変更を実行時に適用できるわけではないことに注意してください。

## 使い方

```conf
keybind = ctrl+shift+r=reload_config
```
