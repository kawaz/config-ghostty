# scroll_page_down

**言語:** [English](../../en/actions/scroll_page_down.en.md) | 日本語

## 説明

画面を1ページ下にスクロールします。

## 使用方法

このアクションは引数なしで呼び出されます：

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/scroll_page_down.ja.md
# 1ページ下にスクロール
keybind page_down = scroll_page_down
```
