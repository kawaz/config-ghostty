# increase_font_size

**言語:** [English](../../en/actions/increase_font_size.en.md) | 日本語

## 説明

フォントサイズをポイント単位 (pt) で指定した量だけ増加させる。

例えば、`increase_font_size:1.5` はフォントサイズを 1.5 ポイント増加させる。

## 使用例

```conf
keybind = super+plus:increase_font_size:1
keybind = super+equal:increase_font_size:2
```
