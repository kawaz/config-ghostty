# new_window

**言語:** [English](../../en/actions/new_window.en.md) | 日本語

## 説明

新しいウィンドウを開きます。

アプリケーションが現在フォーカスされていない場合、これはアプリケーションを前面に表示します。

## 例

```conf
keybind = cmd+n=new_window
keybind = ctrl+alt+n=new_window
```
