# reset_window_size

**Language:** [English](../../en/actions/reset_window_size.en.md) | 日本語

## Description

ウィンドウをデフォルト サイズにリセットします。「デフォルト サイズ」は、新しいウィンドウが作成されるサイズです。ウィンドウが全画面表示の場合、これは効果がありません。

macOS でのみ実装されています。

## Example

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/actions/reset_window_size.ja.md
# ウィンドウをデフォルト サイズにリセット
keybind = super+0=reset_window_size
```
