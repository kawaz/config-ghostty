# toggle_secure_input

**言語:** [English](../../en/actions/toggle_secure_input.en.md) | 日本語

## 説明

セキュア入力モードを切り替えます。

これは、パスワードまたはその他の機密情報を入力するときに、アプリケーションがキーボード入力を監視することを防ぐために使用されます。

これはフォーカスされたターミナルだけでなく、アプリケーション全体に適用されます。無効化するには、手動で切り替えを解除するか、Ghostty 全体を終了する必要があります。

macOS でのみ実装されています。これはビルトイン システム API を使用します。
