# equalize_splits

**言語:** [English](../../en/actions/equalize_splits.en.md) | 日本語

## 説明

現在のウィンドウ内のすべてのスプリットのサイズを均等にします。

## 例

```conf
keybind = alt+e=equalize_splits
```
