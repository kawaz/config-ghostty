# toggle_fullscreen

**言語:** [English](../../en/actions/toggle_fullscreen.en.md) | 日本語

## 説明

現在のウィンドウをフルスクリーン表示またはフルスクリーン表示解除します。

## 例

```conf
keybind = f11=toggle_fullscreen
keybind = cmd+ctrl+f=toggle_fullscreen
```
