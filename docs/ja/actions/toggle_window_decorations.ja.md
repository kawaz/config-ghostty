# toggle_window_decorations

**言語:** [English](../../en/actions/toggle_window_decorations.en.md) | 日本語

## 説明

現在のウィンドウのウィンドウデコレーション（タイトルバー、ボタンなど）を切り替えます。

Linux のみで実装されています。

## 例

```conf
keybind = alt+f11=toggle_window_decorations
```
