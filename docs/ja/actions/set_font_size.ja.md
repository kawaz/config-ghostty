# set_font_size

**言語:** [English](../../en/actions/set_font_size.en.md) | 日本語

## 説明

フォントサイズをポイント単位 (pt) で指定したサイズに設定する。

例えば、`set_font_size:14.5` はフォントサイズを 14.5 ポイントに設定する。

## 使用例

```conf
keybind = super+1:set_font_size:12
keybind = super+2:set_font_size:14
keybind = super+3:set_font_size:16
```
