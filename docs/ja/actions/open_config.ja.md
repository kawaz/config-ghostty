# open_config

**Language:** [English](../../en/actions/open_config.en.md) | 日本語

## 説明

デフォルト OS エディタで設定ファイルを開きます。

デフォルト OS エディタが設定されていない場合、これは失敗します。現在のところ、設定ファイルを開く際の失敗はログにのみ表示されます。

## 使い方

```conf
keybind = ctrl+shift+comma=open_config
```
