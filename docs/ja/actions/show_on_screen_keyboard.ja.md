# show_on_screen_keyboard

**Language:** [English](../../en/actions/show_on_screen_keyboard.en.md) | 日本語

## 説明

画面上のキーボード（存在する場合）を表示します。

Linux（GTK）でのみ実装されています。GNOME では、"スクリーン キーボード" ユーザー補助機能をオンにする必要があります。これは Settings > Accessibility > Typing で確認できます。他のプラットフォームは現在のところテストされていません。

## 使用方法

```
show_on_screen_keyboard
```
