# foreground

**Language:** [English](../../en/config/foreground.en.md) | 日本語

## 説明

ウィンドウの前景色。
16 進数（`#RRGGBB` または `RRGGBB`）または名前付き X11 色として指定します。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/foreground.ja.md
# 前景テキストの色 (default: "#ffffff")
foreground = #ffffff
```
