# クイックターミナルウィンドウのレイヤー。レイヤーが高いほど、
# クイックターミナルが隠す可能性のあるウィンドウが多くなる。
#
# 有効な値は：
#
#  * `overlay`
#
#    クイックターミナルはすべてのウィンドウの前に表示される。
#
#  * `top`（デフォルト）
#
#    クイックターミナルは通常のウィンドウの前に表示されるが、
#    ロック画面などのフルスクリーンオーバーレイの背後に表示される。
#
#  * `bottom`
#
#    クイックターミナルは通常のウィンドウの背後に表示されるが、
#    壁紙および背景レイヤー内の他のウィンドウの前に表示される。
#
#  * `background`
#
#    クイックターミナルはすべてのウィンドウの背後に表示される。
#
# GTK Wayland のみ。
#
# 利用可能：1.2.0 以降
gtk-quick-terminal-layer = top
