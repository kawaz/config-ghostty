# クイックターミナルウィンドウのレイヤー。レイヤーが高いほど、
# クイックターミナルが隠すことができるウィンドウの数が増えます。
#
# 有効な値：
#
#  * `overlay`
#
#    クイックターミナルはすべてのウィンドウの前に表示されます。
#
#  * `top` (デフォルト)
#
#    クイックターミナルはノーマルウィンドウの前に表示されますが、
#    ロック画面などのフルスクリーンオーバーレイの後ろに表示されます。
#
#  * `bottom`
#
#    クイックターミナルはノーマルウィンドウの後ろに表示されますが、
#    壁紙および背景レイヤー内の他のウィンドウの前に表示されます。
#
#  * `background`
#
#    クイックターミナルはすべてのウィンドウの後ろに表示されます。
#
# GTK Wayland のみ。
#
# 利用可能なバージョン: 1.2.0
gtk-quick-terminal-layer = top
