# Linux上で `alt+click`、macOS上で `option+click` を使用してプロンプト時の
# カーソルを移動できる機能を有効にします。
#
# この機能はシェル統合（具体的には `OSC 133` によるプロンプトマーキング）を
# 必要とし、プライマリスクリーンモードでのみ動作します。vim などの
# 代替スクリーンアプリケーションは通常、この機能の独自バージョンを備えていますが、
# この設定はそれを制御しません。
#
# この機能は目的の位置を一連の合成矢印キー操作に変換することで動作するため、
# エッジケースの周辺に奇妙な動作が発生する可能性があります。これは残念ながら、
# ほかに実装方法がないため、すべてのターミナルでこの機能がこのように
# 実装されています。
cursor-click-to-move = true
