# 選択されたテキストをクリップボードに自動的にコピーするかどうか。`true`
# の場合、選択クリップボード (selection clipboard) にコピーすることを
# 優先し、そうでない場合はシステムクリップボードにコピーします。
#
# 値 `clipboard` の場合、選択クリップボードおよびシステムクリップボードの
# 両方にテキストがコピーされます。
#
# 中クリックペースト (middle-click paste) は常に選択クリップボードを使用します。
# 中クリックペーストはこのオプションが `false` であっても常に有効です。
#
# デフォルト値は Linux と macOS で true です。
copy-on-select = true
