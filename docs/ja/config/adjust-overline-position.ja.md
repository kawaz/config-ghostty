# adjust-overline-position

**Language:** [English](../../en/config/adjust-overline-position.en.md) | 日本語

## Description

セルの上部から上線の上部までの距離(ピクセルまたはパーセンテージ調整)。増加させて上線を下に移動、減少させて上線を上に移動します。`adjust-cell-width` での調整に関する注記を参照してください。

## Default

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/adjust-overline-position.ja.md
# 上線テキストの垂直位置を調整 (default: )
adjust-overline-position =
```
