# タブバーを表示するかどうか。
#
# 有効な値：
#
#  - `always`
#
#    タブが 1 つだけの場合でも、常にタブバーを表示。
#
#    利用可能時期: 1.2.0
#
#  - `auto` (デフォルト)
#
#    タブバーを自動的に表示/非表示。タブバーは 2 つ以上のタブがある場合のみ表示される。
#
#  - `never`
#
#    タブバーを表示しない。タブはタブオーバービューまたはキーバインドアクションを
#    通じてのみアクセス可能。
#
# 現在 Linux (GTK) でのみサポートされている。
window-show-tab-bar = auto
