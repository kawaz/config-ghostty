# タブバーを表示するかどうか。
#
# 有効な値：
#
#  - `always`
#
#    タブが1つだけの場合でも、常にタブバーを表示します。
#
#    利用可能なバージョン: 1.2.0
#
#  - `auto` *(デフォルト)*
#
#    タブバーを自動的に表示・非表示にします。タブバーは
#    2つ以上のタブが存在する場合にのみ表示されます。
#
#  - `never`
#
#    タブバーを表示しません。タブはタブオーバービュー
#    またはキーバインドアクションでのみアクセス可能です。
#
# 現在のところLinux (GTK)でのみサポートされています。
window-show-tab-bar = auto
