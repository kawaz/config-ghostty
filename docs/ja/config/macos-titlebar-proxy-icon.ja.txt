# macOS のタイトルバーに表示されるプロキシアイコン(Proxy Icon)の表示可否を制御します。
# プロキシアイコンは、現在の作業ディレクトリを表すフォルダのアイコンです。
# macOS 標準ターミナル(Terminal.app)のタイトルバーで、これを非常に明確に
# 確認できます。
#
# プロキシアイコンは、ネイティブ macOS タイトルバースタイルの場合のみ
# 表示されます。
#
# 有効な値は以下の通りです：
#
#   * `visible` - プロキシアイコンを表示します。
#   * `hidden` - プロキシアイコンを非表示にします。
#
# デフォルト値は `visible` です。
#
# この設定は実行時に変更でき、すべてのオープン中のウィンドウに影響を
# 与えます。ただし、ウィンドウの作業ディレクトリが再度変更された後に
# のみ反映されます。したがって、この設定変更後に機能させるには、通常は
# `cd` で別のディレクトリに移動したり、エディタで別のファイルを開いたり
# する必要があります。
macos-titlebar-proxy-icon = visible
