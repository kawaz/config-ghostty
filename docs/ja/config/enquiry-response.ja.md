# enquiry-response

**言語:** [English](../../en/config/enquiry-response.en.md) | 日本語

## 説明

実行しているコマンドから `ENQ` （0x05）を受け取ったときに送信する文字列。設定しない場合のデフォルトは空文字列です。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/enquiry-response.ja.md
# ENQ制御文字を受け取ったときに送信する文字列 (default: )
enquiry-response =
```
