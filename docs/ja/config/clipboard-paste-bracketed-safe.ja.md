# clipboard-paste-bracketed-safe

**言語:** [English](../../en/config/clipboard-paste-bracketed-safe.en.md) | 日本語

## 説明

true の場合、括弧付きペーストは安全なものと見なされます。デフォルトでは、括弧付きペーストは安全なものと見なされます。「括弧付き」ペーストは、実行中のプログラムが括弧付きペーストモードを有効にしている間に行われるペーストです（これはターミナルエミュレータではなく、実行中のプログラムによって設定される設定です）。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/clipboard-paste-bracketed-safe.ja.md
# 括弧付きペーストを安全なものとして扱う (default: true)
clipboard-paste-bracketed-safe = true
```
