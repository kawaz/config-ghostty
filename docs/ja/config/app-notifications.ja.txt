# Ghostty が表示するアプリ内通知を制御します。
#
# Linux (GTK) では、アプリ内通知はトーストとして表示されます。トーストは
# ターミナルウィンドウの上に重ねて表示されます。これらは重要ではないが
# 重要な可能性のある情報を表示するために使用されます。
#
# 利用可能な通知は以下の通りです:
#
#   - `clipboard-copy` (デフォルト: true) - テキストがクリップボードに
#     コピーされたときに通知を表示します。
#   - `config-reload` (デフォルト: true) - 設定が再読み込みされたときに
#     通知を表示します。
#
# 通知を有効にするには、その通知の名前を指定します。
# 通知を無効にするには、名前の前に `no-` をつけてください。例えば、
# `clipboard-copy` を無効にするには、この設定を `no-clipboard-copy` に
# 設定します。有効にするには、`clipboard-copy` に設定します。
#
# 複数の通知は、カンマで区切って有効または無効にできます。
#
# "false" の値はすべての通知を無効にします。"true" の値はすべての
# 通知を有効にします。
#
# この設定は GTK にのみ適用されます。
#
# 利用可能なバージョン: 1.1.0
app-notifications = clipboard-copy,config-reload
