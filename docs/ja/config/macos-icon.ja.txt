# macOSアプリアイコンをカスタマイズします。
#
# これはDockやアプリケーションスイッチャーなどに表示されるアイコンにのみ
# 影響します。Finderのアイコンには影響しません。なぜなら、そのアイコンは
# 署名済みアプリケーションバンドル内のハードコードされた値で制御されており、
# 実行時に変更することはできないからです。正確に何が影響を受けるかについて
# の詳細は、`NSApplication.icon` Apple ドキュメントを参照してください。
# これがアイコンの設定に使用されているAPIです。
#
# 有効な値:
#
#  * `official` - 公式Ghosttyアイコンを使用します。
#  * `blueprint`, `chalkboard`, `microchip`, `glass`, `holographic`,
#    `paper`, `retro`, `xray` - アーティストが手作業で作成した公式の
#    Ghosttyアイコンのバリアント（AIなし）。
#  * `custom` - 完全にカスタムアイコンを使用します。場所は追加の
#    `macos-custom-icon` 設定を使用して指定する必要があります。
#  * `custom-style` - 公式Ghosttyアイコンを使用しますが、さまざまな
#    レイヤーにカスタムスタイルが適用されます。カスタムスタイルは
#    追加の `macos-icon` プレフィックスの設定を使用して指定する必要があります。
#    このスタイルでは `macos-icon-ghost-color` と `macos-icon-screen-color`
#    の設定が必須です。
#
# 警告: `custom-style` オプションは _実験的_ です。カスタムスタイルの形式が
# 将来変更される可能性があります。利用可能なレイヤーとカスタマイズ
# オプションの正確な仕様をまだ最終決定中です。
#
# その他の注意事項:
#
#   * 更新ダイアログのアイコンは常に公式アイコンです。これは
#     更新ダイアログが別のフレームワークで管理されており、
#     大きな労力をかけずにはカスタマイズできないためです。
macos-icon = official
