# 機能設定の構文は以下の通りです。`feat` はフォント機能です：
#

#   * フォント機能の値を設定します。例えば `feat=2`、`feat = 3`、`feat 4`
#   * フォント機能名をクォートで囲むことができます。これは、このconfig
