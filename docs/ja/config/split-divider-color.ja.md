# split-divider-color

**言語:** [English](../../en/config/split-divider-color.en.md) | 日本語

## 説明

分割区切り線の色。設定されていない場合は、デフォルトが選択される。16 進数 (`#RRGGBB` または `RRGGBB`) または名前付き X11 色として指定。

利用可能時期: 1.1.0

## デフォルト

```conf
# https://ghostty.org/docs/config/reference#split-divider-color
# 分割区切り線の色 (デフォルト: (プラットフォーム依存))
split-divider-color =
```
