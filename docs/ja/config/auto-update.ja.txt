# Ghosttyの自動更新機能を制御します。現在のところmacOSのみサポートされており、
# LinuxビルドはGhosttyによって中央管理されていないパッケージマネージャーを
# 経由して配布されているためです。
#
# 更新の確認またはダウンロードは、基盤となるプロトコルによって義務付けられた
# 標準的なネットワーク情報以外の情報をプロジェクトに送信しません。別の言い方をすれば
# Ghosttyは更新プロセスに明示的なトラッキングを追加していません。更新プロセスは
# 最新バージョンに関する情報をダウンロードし、クライアント側で現在のバージョンと
# 比較することで機能します。
#
# 有効な値：
#
#  * `off` - 自動更新を無効にします。
#  * `check` - 更新を確認し、更新が利用可能な場合はユーザーに通知しますが、
#    更新を自動的にダウンロードまたはインストールしません。
#  * `download` - 更新を確認し、更新を自動的にダウンロードしてユーザーに通知しますが、
#    更新を自動的にインストールしません。
#
# 設定しない場合、Sparkleのデフォルト動作に従い、標準ユーザーデフォルト
# （`defaults(1)`）に保存されたプリファレンスを尊重します。
#
# 実行時にこの値を変更することは、小さな遅延後に機能します。
