# maximize

**言語:** [English](../../en/config/maximize.en.md) | 日本語

## 説明

新しいウィンドウを最大化状態で開始するかどうかを設定します。この設定は新しいウィンドウに適用され、タブ、スプリット等には適用されません。ただし、この設定は最初のウィンドウだけでなく、すべての新しいウィンドウに適用されます。

利用可能：1.1.0 以降

## デフォルト値

```conf
# https://ghostty.org/docs/config/reference#maximize
# 新しいウィンドウを最大化状態で開始 (default: false)
maximize = false
```
