# 明示的な背景色が設定されたセルに背景の透明度を適用します。
#
# 通常、`background-opacity` はウィンドウ背景にのみ適用されます。
# セルに赤などの明示的な背景色が設定されている場合、その背景色は
# 完全に不透明になります。この結果、Neovim や Tmux のような
# ターミナルの背景色を再描画するターミナルアプリケーションは、
# `background-opacity` を尊重しない可能性があります（設計上の理由です）。
#
# これを `true` に設定すると、セルが明示的な背景色を設定しているかどうかに
# 関係なく、`background-opacity` がすべてのセルに適用されます。
#
# 利用可能なバージョン: 1.2.0
background-opacity-cells = false
