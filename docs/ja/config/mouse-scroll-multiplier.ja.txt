# マウスホイールでのスクロール距離の乗数。
#
# `precision:` または `discrete:` というプレフィックスを使用して、特定の
# タイプのデバイスでのスクロールのみに乗数を設定できます。これらはカンマで
# 区切ることで、両方のタイプの乗数を同時に設定できます。例：
# `precision:0.1,discrete:3`。プレフィックスを使用しない場合、乗数はすべての
# スクロールデバイスに適用されます。プレフィックスの指定は Ghostty 1.2.1 で
# 導入されました。
#
# この値は [0.01, 10,000] の範囲に制限されます。これらはいずれも極端な値であり、
# どちらかの極端な値を設定すると、悪い使用体験を得ることになります。
#
# デフォルト値は、離散デバイスでは "3"、精密デバイスでは "1" です。
mouse-scroll-multiplier = precision:1,discrete:3
