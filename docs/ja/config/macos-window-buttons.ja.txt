# macOSのタイトルバーにあるウィンドウボタンが表示されるかどうか。ウィンドウ
# ボタンは、ほとんどのmacOSアプリケーションの左上隅にある色付きボタンで、
# トラフィックライト（traffic lights）とも呼ばれ、ウィンドウを閉じたり、
# 最小化したり、ズームしたりすることができます。
#
# `window-decoration = false` または `macos-titlebar-style = hidden` の場合、
# このセッティングは効果がありません。これらのモードではウィンドウボタンは
# 常に隠れます。
#
# 有効な値は以下のとおりです：
#
#   * `visible` - ウィンドウボタンを表示します。
#   * `hidden` - ウィンドウボタンを隠します。
#
# デフォルト値は `visible` です。
#
# このオプションを実行時に変更すると、新しいウィンドウにのみ適用されます。
#
# 利用可能：1.2.0 以降
macos-window-buttons = visible
