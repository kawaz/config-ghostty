# gtk-quick-terminal-namespace

**Language:** [English](../../en/config/gtk-quick-terminal-namespace.en.md) | 日本語

## 説明

クイックターミナルウィンドウの名前空間。

これは、Wayland コンポジタおよび/またはスクリプトによってレイヤーサーフェスのタイプを決定し、ユニークなエフェクトを適用する可能性があるために使用される識別子です。

GTK Wayland のみです。

利用可能になったのは: 1.2.0

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/gtk-quick-terminal-namespace.ja.md
# クイックターミナルウィンドウの名前空間識別子 (default: ghostty-quick-terminal)
gtk-quick-terminal-namespace = ghostty-quick-terminal
```
