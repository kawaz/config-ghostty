# scrollback-limit

**言語:** [English](../../en/config/scrollback-limit.en.md) | 日本語

## 説明

スクロールバックバッファのサイズ (バイト単位)。これはアクティブスクリーンも含む。これが何に設定されていても、表示画面に十分なメモリが常に割り当てられ、残りはスクロールバックの制限。

この制限に達すると、古い行がスクロールバックから削除される。

スクロールバックは現在、完全にメモリに存在する。つまり、この値が大きいほど、潜在的なメモリ使用量が大きくなる。スクロールバックはこの制限まで遅延割り当てされるため、これを非常に大きい値に設定した場合、すぐに多くのメモリを消費することはない。

このサイズは、アプリケーション全体ではなく、ターミナルサーフェスごとである。

現在、無制限のスクロールバックバッファを設定することは不可能。これは今後予定されている機能。

これは実行時に変更できますが、新しいターミナルサーフェスにのみ影響を与える。

## デフォルト

```conf
# https://ghostty.org/docs/config/reference#scrollback-limit
# スクロールバックバッファの最大サイズ (バイト単位) (デフォルト: 10000000)
scrollback-limit = 10000000
```
