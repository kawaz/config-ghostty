# custom-shader-animation

**言語:** [English](../../en/config/custom-shader-animation.en.md) | 日本語

## 説明

`true` （デフォルト）の場合、カスタム シェーダーが使用されている場合、フォーカスされたターミナル サーフェスはアニメーション ループを実行します。これはやや CPU 使用量を増やしますが（通常は 10% 未満）、シェーダーをアニメーション化できます。これはカスタム シェーダーがあり、ターミナルがフォーカスされている場合にのみ実行されます。

これが `false` に設定されている場合、ターミナルとカスタム シェーダーはターミナルが更新されたときのみレンダリングされます。これはより効率的ですが、シェーダーはアニメーション化されません。

これは `always` に設定することもでき、ターミナルがフォーカスされているかどうかに関わらず、アニメーション ループは常に実行されます。アニメーション ループは、カスタム シェーダーが使用されている場合にのみ実行されます。これはターミナル サーフェスあたり CPU 使用量が増加し、シェーダーとターミナル使用方法に応じて非常にコストになる可能性があることに注意してください。

これは実行時に変更でき、すべての開いているターミナルに影響を与えます。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/custom-shader-animation.ja.md
# カスタム シェーダーのアニメーション ループを実行する (default: true)
custom-shader-animation = true
```
