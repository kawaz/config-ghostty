# `true` （デフォルト）の場合、カスタム シェーダーが使用されている場合、
# フォーカスされたターミナル サーフェスはアニメーション ループを実行します。
# これはやや CPU 使用量を増やしますが（通常は 10% 未満）、シェーダーを
# アニメーション化できます。これはカスタム シェーダーがあり、
# ターミナルがフォーカスされている場合にのみ実行されます。
#
# これが `false` に設定されている場合、ターミナルとカスタム シェーダーは
# ターミナルが更新されたときのみレンダリングされます。これはより効率的ですが、
# シェーダーはアニメーション化されません。
#
# これは `always` に設定することもでき、ターミナルがフォーカスされているかどうかに
# 関わらず、アニメーション ループは常に実行されます。
# アニメーション ループは、カスタム シェーダーが使用されている場合にのみ
# 実行されます。これはターミナル サーフェスあたり CPU 使用量が増加し、
# シェーダーとターミナル使用方法に応じて非常にコストになる可能性があることに
# 注意してください。
#
# これは実行時に変更でき、すべての開いているターミナルに影響を与えます。
custom-shader-animation = true
