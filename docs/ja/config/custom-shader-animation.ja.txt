# `true`（デフォルト）の場合、フォーカスされたターミナルサーフェスは
# カスタムシェーダーが使用されている時にアニメーションループを実行します。
# これはわずかに多くのCPUを使用しますが（一般的に10%未満）、シェーダーが
# アニメーションすることを可能にします。これはカスタムシェーダーが
# ある場合にのみ実行され、ターミナルがフォーカスされている場合です。
#
# これを`false`に設定した場合、ターミナルとカスタムシェーダーは
# ターミナルが更新された時にのみレンダリングされます。
# これはより効率的ですが、シェーダーはアニメーションしません。
#
# これを`always`に設定することもでき、ターミナルがフォーカスされて
# いるかどうかに関わらず常にアニメーションループを実行します。
# アニメーションループはカスタムシェーダーが使用されている場合にのみ
# 実行されます。これはターミナルサーフェスごとにより多くのCPUを
# 使用し、シェーダーとターミナルの使用方法によっては非常に
# 負荷になる可能性があります。
#
# これは実行時に変更でき、すべてのオープンなターミナルに影響します。
custom-shader-animation = true
