# 検索されます。設定されていない場合は、システムからデフォルトが取得されます。
# デフォルト取得のルール：
#
#   * `SHELL` 環境変数
#
#   * `passwd` エントリ（ユーザー情報）
#
# 追加の引数を含めることができます。追加の引数が指定された場合、
# コマンドは `/bin/sh -c` を使用して実行され、シェル引数展開が行われます。
#
# シェル展開を完全に回避するには、コマンドの前に `direct:` を付けます。
# 例：`direct:nvim foo`。これにより `/bin/sh` へのラウンドトリップは回避されますが、
# スペースを含む引数、`~` を含むファイルパス、グロブなどのシェル解析
# はサポートされません。（1.2.0 以降で利用可能）
#
# また、コマンドの前に `shell:` を明示的に付けて、常にコマンドをシェルで
# ラップすることもできます。ヒューリスティクスが間違っている場合に
# 正しいモードを確実にするために使用できます。（1.2.0 以降で利用可能）
#
# このコマンドは、新しいウィンドウやタブなど、すべての新しい
