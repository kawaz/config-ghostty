# 前景色と背景色の最小コントラスト比。コントラスト比は 1 から 21 の間の値である。
# 1 の値はコントラストなし（例：黒の上の黒）を意味する。この値は
# [WCAG 2.0 仕様](https://www.w3.org/TR/WCAG20/)で定義されたコントラスト比である。
#
# 見えないテキスト（背景と同じ色）を避けたい場合、1.1 の値が良い値である。
# 読みにくいテキストを避けたい場合、3 以上の値が良い値である。
# 値が高いほど、テキストが黒または白になる可能性が高くなる。
#
# この値は絵文字または画像には適用されない。
minimum-contrast = 1
