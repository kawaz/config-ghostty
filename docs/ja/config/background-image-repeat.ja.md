# background-image-repeat

**言語:** [English](../../en/config/background-image-repeat.en.md) | 日本語

## 説明

背景画像を繰り返すかどうか。

これが true に設定されている場合、背景画像が完全にターミナル領域を埋めていないために空白スペースが発生する場合、背景画像が繰り返されます。

デフォルト値は `false` です。

利用可能: 1.2.0 以降

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/background-image-repeat.ja.md
# 背景画像を繰り返してターミナルを埋める (default: false)
background-image-repeat = false
```
