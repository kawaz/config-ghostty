# サーフェースを下部にスクロールするタイミング。フォーマットはコンマで区切られた
# オプションのリストです。オプションの前に `no-` を付けると無効になります。
# オプションを省略した場合は、デフォルト値が使用されます。
#
# 利用可能なオプション:
#
# - `keystroke` 有効にすると、ユーザーが PTY にデータを送信するキーを押した場合
#   （基本的には修飾キーまたは Ghostty で処理されるキーバインド以外の場合）
#   にサーフェースを下部にスクロールします。
#
# - `output` 有効にすると、表示する新しいデータがある場合にサーフェースを
#   下部にスクロールします。（現在は未実装です。）
#
# デフォルトは `keystroke, no-output` です。
scroll-to-bottom = keystroke,no-output
