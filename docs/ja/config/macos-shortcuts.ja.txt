# macOS ショートカットが Ghostty を制御することを許可するかどうか
#
# Ghostty は、ショートカットが Ghostty を制御および操作できるようにする
# 多くのアクションを公開しています。これには、新しいターミナルの作成、
# ターミナルへのテキスト送信、コマンド実行、任意のキーバインド
# アクションの呼び出しなどが含まれます。
#
# これは強力な機能ですが、悪意のあるショートカットがインストールされ
# 実行される可能性がある場合、セキュリティ上のリスクになる可能性があります。
# したがって、この設定によってこの機能を無効にすることができます。
#
# 有効な値は以下の通りです：
#
# * `ask` - ユーザーに権限を求めるかどうかを質問します。Ghostty はこの
#   選択を記憶し、二度と質問しません。これはマイク アクセスやカメラ
#   アクセスなど、他の macOS 権限と同様です。
#
# * `allow` - 質問することなく、ショートカットで Ghostty を制御することを
#   許可します。
#
# * `deny` - ショートカットが Ghostty を制御することを拒否します。
#
# 利用開始バージョン: 1.2.0
macos-shortcuts = ask
