# window-position-x

**言語:** [English](../../en/config/window-position-x.en.md) | 日本語

## 説明

ウィンドウの開始位置を設定します。この位置はピクセル単位で、プライマリ モニターの左上隅を基準にしています。効果を持たせるには両方の値を設定する必要があります。1 つの値だけが設定されている場合、それは無視されます。

ウィンドウマネージャーは位置に制限を設けたり、位置をオーバーライドしたりする可能性があることに注意してください。たとえば、タイリングウィンドウマネージャーは、グリッド内に適合するようにウィンドウを特定の位置にするように強制する可能性があります。Ghostty がこれについて何かできることはありませんが、努力するでしょう。

また、負の値はオペレーティング システムとウィンドウマネージャーの判断によるものであることに注意してください。一部のウィンドウマネージャーは、ウィンドウを画面外に配置することを許可しない可能性があります。

無効な位置はランタイム固有ですが、一般的に位置は最も近い有効な位置にクランプされます。

macOS では、ウィンドウの位置は表示画面領域の左上隅を基準にしています。これは、メニューバーが表示されている場合、ウィンドウはメニューバーの下に配置されることを意味します。

注：これは macOS でのみサポートされています。GTK ランタイムはウィンドウの位置設定をサポートしていません。ウィンドウは X11 でのみ自身を配置でき、Wayland では配置できないためです。
