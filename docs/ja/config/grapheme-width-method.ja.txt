# グラフェムクラスタのセル幅を計算するために使用するメソッド。
# デフォルト値は `unicode` で、Unicode標準を使用してグラフェム幅を
# 決定します。これにより正確なグラフェム幅が得られますが、シェルなど
# `wcswidth` のような従来のメソッドを使用する可能性があるプログラムでは
# カーソルの同期がズレる問題が発生する場合があります。
#
# 有効な値は以下の通りです：
#
# * `legacy` - wcswidth のような従来のメソッドを使用して
#   グラフェム幅を決定します。従来のプログラムとの互換性が最大化されますが、
#   スキントーン絵文字、非英語文字などの特定のグラフェムでは
#   グラフェム幅が不正確になる場合があります。
#
#   これが「レガシー」と呼ばれるのは、動作が未定義であり、
#   将来的にそれを変更する能力を保持したいためです。
#   例えば、今後libc の `wcswidth` を使用するか使用しないかは未定です。
#
# * `unicode` - Unicode標準を使用してグラフェム幅を決定します。
#
# 実行中のプログラムがターミナルモード2027を明示的に有効にした場合、
# この設定に関係なく `unicode` 幅が強制されます。モード2027が
# リセットされると、この設定が再度使用されます。
#
# この設定は実行時に変更できますが、既存のターミナルには影響しません。
# 新しいターミナルのみが新しい設定を使用します。
grapheme-width-method = unicode
