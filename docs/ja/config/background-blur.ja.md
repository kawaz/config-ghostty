# background-blur

**言語:** [English](../../en/config/background-blur.en.md) | 日本語

## 説明

`background-opacity` が 1 より小さい場合に、背景をぼかすかどうかを設定します。

有効な値は以下の通りです：

- ぼかしの強度を指定する負でない整数
- `false`。ぼかし強度 0 に相当する
- `true`。デフォルトのぼかし強度 20 に相当し、見た目の良いぼかしに適している。より高いぼかし強度を使用するとレンダリングの問題やパフォーマンスの問題が発生する可能性がある。

macOS と一部の Linux デスクトップ環境でサポートされています：

- KDE Plasma（Wayland と X11）

警告：KDE Plasma では正確なぼかし強度は無視されます。この設定を `true` に設定するか、正の任意のぼかし強度値を使用すると、同じ効果が得られます。理由は、Plasma を駆動するウィンドウコンポジター KWin にはグローバルぼかし設定が 1 つしかなく、アプリケーションが個別のぼかし設定を指定することを許可しないためです。

KWin のグローバルぼかし設定を構成するには、システム設定を開き、「アプリケーションとウィンドウ」>「ウィンドウ管理」>「デスクトップ効果」に移動して、「ぼかし」プラグインを選択します。無効化されている場合は、左側のチェックボックスをチェックして有効化します。次に「構成」ボタンをクリックすると、Ghostty を含むすべてのアプリケーションの背景ぼかしとノイズ強度を設定することができる 2 つのスライダーが表示されます。

その他のすべての Linux デスクトップ環境は現在のところサポートされていません。背景ぼかしをサポートするには、ユーザーが環境固有の設定を行うか、または統一されたインターフェースがないため、サードパーティプラグインをインストールする必要があります。

## デフォルト値

```conf
# https://ghostty.org/docs/config/reference#background-blur
# 背景ぼかし効果の有効/無効を設定 (default: false)
background-blur = false
```
