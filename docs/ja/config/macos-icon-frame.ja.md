# macos-icon-frame

**Language:** [English](../../en/config/macos-icon-frame.en.md) | 日本語

## 説明

macOS アプリ アイコンのフレームに使用するマテリアル。

有効な値：

- `aluminum` - ブラッシュド アルミニウム フレーム。これはデフォルトです。
- `beige` - クラシックな 90 年代のコンピューター ベージュ フレーム。
- `plastic` - 光沢のある濃いプラスチック フレーム。
- `chrome` - 光沢のあるクロム フレーム。

注：この設定は、`macos-icon` が `custom-style` に設定されている場合に必要です。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/macos-icon-frame.ja.md
# macOS アプリアイコンフレームのマテリアル仕上げ (default: aluminum)
macos-icon-frame = aluminum
```
