# 垂直ウィンドウパディング。これはターミナルセルとウィンドウの上下の枠線の間に
# パディングを適用します。値はポイント単位です。つまり、画面の DPI に応じて
# 適切にスケールされます。
#
# この値が大きすぎる場合、パディングによってグリッドが完全に圧縮されるため、
# 画面には何もレンダリングされません。合理的な値を選択するのはあなた次第です。
# 不合理な値を選択した場合、ログに警告が表示されます。
#
# この設定を実行時に変更すると、新しいターミナル（新しいウィンドウ、タブなど）
# にのみ影響します。
#
# 異なる上部と下部のパディングを設定するには、コンマで区切った 2 つの数値を
# 指定します。たとえば、`window-padding-y = 2,4` は、上部パディングを 2、
# 下部パディングを 4 に設定します。両方のパディングを同じ値に設定する場合は、
# 単一の値を使用できます。たとえば、`window-padding-y = 2` は両方のパディングを 2 に設定します。
window-padding-y = 2
