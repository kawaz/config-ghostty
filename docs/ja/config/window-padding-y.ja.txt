# 垂直方向のウィンドウパディング。これはターミナルセルとウィンドウの上下の
# 境界線の間にパディングを適用します。値はポイント単位であり、スクリーン DPI
# に応じて適切にスケーリングされます。
#
# この値が大きすぎると、グリッドがパディングによって完全に圧縮されるため、
# スクリーンに何も表示されません。ユーザーが適切な値を選択することが重要です。
# 不適切な値を選択した場合、ログに警告が表示されます。
#
# この設定をランタイム時に変更すると、新しいターミナルにのみ影響します。
# つまり、新しいウィンドウ、タブなどが対象になります。
#
# 上下のパディングを異なる値に設定するには、カンマで区切られた 2 つの
# 数値を指定します。例えば、`window-padding-y = 2,4` は上のパディングを 2 に、
# 下のパディングを 4 に設定します。両方のパディングを同じ値に設定したい場合は、
# 単一の値を使用できます。例えば、`window-padding-y = 2` は両方のパディングを 2 に設定します。
window-padding-y = 2
