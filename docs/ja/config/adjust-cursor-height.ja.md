# adjust-cursor-height

**言語:** [English](../../en/config/adjust-cursor-height.en.md) | 日本語

## 説明

カーソルの高さのピクセルまたはパーセンテージ調整。現在、すべてのカーソルタイプに適用されます：bar、rect、outline rect。`adjust-cell-width` の調整に関する注意を参照してください。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/adjust-cursor-height.ja.md
# カーソル高さの調整 (default: unset)
adjust-cursor-height =
```
