# gtk-toolbar-style

**言語:** [English](../../en/config/gtk-toolbar-style.en.md) | 日本語

## 説明

トップバーとボトムバーのタブバーの外観を決定します。

有効な値は:

* `flat` - トップバーとボトムバーはターミナルウィンドウと平らです。
* `raised` - トップバーとボトムバーはターミナル領域に影を落とします。
* `raised-border` - `raised` に似ていますが、影はより微妙なボーダーに置き換わります。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/gtk-toolbar-style.ja.md
# GTK ツールバーとタブバーの外観 (default: raised)
gtk-toolbar-style = raised
```
