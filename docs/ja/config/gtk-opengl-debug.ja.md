# gtk-opengl-debug

**言語:** [English](../../en/config/gtk-opengl-debug.en.md) | 日本語

## 説明

GTK の OpenGL デバッグログを有効または無効にする。デフォルトはデバッグビルドの場合 `true`、その他は `false`。

利用可能時期: 1.1.0

## デフォルト

```conf
# https://ghostty.org/docs/config/reference#gtk-opengl-debug
# GTK OpenGL デバッグログを有効または無効にする (デフォルト: false)
gtk-opengl-debug = false
```
