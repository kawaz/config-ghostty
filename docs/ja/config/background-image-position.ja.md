# background-image-position

**言語:** [English](../../en/config/background-image-position.en.md) | 日本語

## 説明

背景画像の位置。

有効な値:
- `top-left`
- `top-center`
- `top-right`
- `center-left`
- `center`
- `center-right`
- `bottom-left`
- `bottom-center`
- `bottom-right`

デフォルト値は `center` です。

利用可能: 1.2.0 以降

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/background-image-position.ja.md
# 背景画像の位置 (default: center)
background-image-position = center
```
