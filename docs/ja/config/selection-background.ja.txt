# タイピング時に選択テキストをクリアするかどうか。デフォルトは`true`。
# これはほとんどのターミナルエミュレータおよびテキスト入力フィールドの
# 標準的な動作です。これを`false`に設定した場合、選択テキストは
# タイピング時にクリアされません。
#
# "タイピング"は具体的には、ターミナル内で実行されているアプリケーション
# （例：シェル）に送信されるデータを生成する、修飾キー（Shift、Ctrl、
# Altなど）以外のキープレスと定義されます。さらに、プリエディット（preedit）
# または合成状態が開始されたとき（例：日本語などの言語をタイピングするとき）に
# 選択がクリアされます。
#
# これが`false`の場合、選択は1回のクリックまたは`escape`キーを押すことで
# 手動でクリアできます。
#
# 利用可能なバージョン: 1.2.0
selection-clear-on-typing = true
