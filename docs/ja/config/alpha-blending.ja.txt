# アルファブレンディング時に使用するカラースペース。
#
# テキストと透明度を持つあらゆる画像の表現に影響します。
# さらに、カスタムシェーダーは設定されたカラースペースで色を受け取ります。
#
# macOSではデフォルトは `native`、他のすべてのプラットフォームではデフォルトは
# `linear-corrected` です。
#
# 有効な値：
#
# * `native` - OSのネイティブなカラースペースでアルファブレンディングを実行します。
#   macOSではこれはDisplay P3に対応し、LinuxではsRGBです。
#
# * `linear` - 線形空間（リニアスペース）でアルファブレンディングを実行します。これにより
#   特定のカラーの組み合わせ（例：赤/緑）を使用する場合に非常に目立つテキストの
#   エッジ周辺の暗くなるアーティファクトが排除されますが、ダークテキストが通常より
#   はるかに細く見え、ライトテキストがはるかに太く見えます。
#   これは「ガンマ補正」として知られることもあります。
#
# * `linear-corrected` - `linear` と同じですが、テキストに補正ステップを適用して
#   `native` とほぼまたは完全に同じに見えるようにしながら、暗くなるアーティファクトは
#   一切ありません。
#
# 対応バージョン: 1.1.0
alpha-blending = native
