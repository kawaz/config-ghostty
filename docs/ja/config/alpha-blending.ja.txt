# アルファ合成を実行するときに使用する色空間。
#
# これは、テキストと透明度を持つ任意の画像の外観に影響する。
# さらに、カスタムシェーダーは設定された色空間で色を受け取る。
#
# macOS ではデフォルトは `native`、その他のすべてのプラットフォームでは
# デフォルトは `linear-corrected` である。
#
# 有効な値：
#
# * `native` - OS のネイティブ色空間でアルファ合成を実行する。
#   macOS ではこれは Display P3 に対応し、Linux では sRGB である。
#
# * `linear` - 線形空間でアルファ合成を実行する。これにより、特定の
#   色の組み合わせが使用される場合に非常に目立つテキストのエッジ周辺の
#   暗くなるアーティファクトが除去されるが、暗いテキストは通常よりはるかに
#   薄く見え、明るいテキストはより厚く見える。これは「ガンマ補正」として
#   知られることもある。
#
# * `linear-corrected` - `linear` と同じだが、テキストの補正ステップが
#   適用されており、`native` とほぼまたは完全に同じに見えるが、
#   暗くなるアーティファクトはない。
#
# 利用可能：1.1.0 以降
alpha-blending = native
