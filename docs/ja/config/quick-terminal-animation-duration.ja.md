# quick-terminal-animation-duration

**言語:** [English](../../en/config/quick-terminal-animation-duration.en.md) | 日本語

## 説明

クイック ターミナルの表示と終了アニメーションの期間（秒単位）。0 に設定すると、アニメーションを完全に無効化します。これは実行時に変更できます。

macOS でのみ実装されています。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/quick-terminal-animation-duration.ja.md
# クイック ターミナルアニメーションの期間 (default: 0.2)
quick-terminal-animation-duration = 0.2
```
