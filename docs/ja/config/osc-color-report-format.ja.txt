# OSC シーケンスで色情報をリクエストする場合のレポート形式を設定します。
# Ghostty は現在 OSC 10 (前景色)、OSC 11 (背景色)、および OSC 4 (256色パレット)
# クエリをサポートしており、デフォルトではレポートされる値は拡大縮小されたRGB値です。
# 各コンポーネントは16ビットです。これはほとんどのターミナルがこれらの値をレポートする方法です。
# ただし、一部のレガシーアプリケーションでは8ビット、拡大縮小されていないコンポーネントが
# 必要な場合があります。レポート機能をすべてオフにすることもサポートしています。
# コンポーネントは小文字の16進数値です。
#
# 許可される値は：
#
#   * `none` - OSC 4/10/11 クエリは応答を受け取りません
#
#   * `8-bit` - カラーコンポーネントは拡大縮小なしで返されます（例：`rr/gg/bb`）
#
#   * `16-bit` - カラーコンポーネントは拡大縮小して返されます（例：`rrrr/gggg/bbbb`）
#
# デフォルト値は `16-bit` です。
osc-color-report-format = 16-bit
