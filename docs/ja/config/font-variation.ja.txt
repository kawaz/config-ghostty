# 可変フォント（variable font）は、通常 `-VF.ttf` または `-VF.otf` で終わるファイル名を持つ
# 単一のフォントで、太さ（weight）や傾き（slant）などの設定可能な軸を1つ以上含みます。
# すべてのフォントが可変に対応しているわけではなく、可変フォントであることを明示的に述べている
# フォントのみが動作します。
#
# フォーマットは `id=value` で、`id` は軸識別子（axis identifier）です。軸識別子は常に
# `wght` のような4文字の文字列です。サポートされている軸のリストを取得するには、
# フォントのドキュメントを参照するか、フォント検査ツールを使用してください。
#
# 無効なidと値は通常無視されます。例えば、フォントが重さ100～700のみをサポートしている場合、
# `wght=800` を設定しても何もしません（700に切り詰められることはありません）。
# サポートされている値を確認するには、フォントのドキュメントを参照する必要があります。
#
