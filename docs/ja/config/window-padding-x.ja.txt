# ウィンドウの水平パディング。ターミナルセルと左右のウィンドウボーダーの間に
# パディングを適用します。値はポイント単位で、スクリーンのDPIに応じて
# 適切にスケーリングされます。
#
# この値が大きすぎる場合、グリッドがパディングで完全に圧縮されるため、
# 画面に何も表示されなくなります。ユーザー自身で合理的な値を選択する必要があります。
# 不適切な値を選択した場合、ログに警告が表示されます。
#
# この設定を実行時に変更すると、新しいターミナルのみに反映されます。
# つまり、新しいウィンドウ、タブなどに適用されます。
#
# 左右のパディングを異なる値に設定するには、カンマで区切られた2つの数値を
# 指定します。例えば、`window-padding-x = 2,4` は左パディングを2、
# 右パディングを4に設定します。両方のパディングを同じ値に設定したい場合は、
# 単一の値を使用できます。例えば、`window-padding-x = 2` は両方のパディングを2に設定します。
window-padding-x = 2
