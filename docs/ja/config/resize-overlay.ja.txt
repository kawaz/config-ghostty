# これはリサイズオーバーレイを表示するタイミングを制御します。リサイズオーバーレイは、
# サーフェスのサイズ変更中にターミナルのサイズを表示する一時的なポップアップです。
# 可能なオプションは以下の通りです：
#
#   * `always` - リサイズオーバーレイを常に表示します。
#   * `never` - リサイズオーバーレイを表示しません。
#   * `after-first` - サーフェスが最初に作成される際はリサイズオーバーレイを表示
#                     しませんが、その後のサーフェスのリサイズ時には表示されます。
#
# デフォルトは `after-first` です。
resize-overlay = after-first
