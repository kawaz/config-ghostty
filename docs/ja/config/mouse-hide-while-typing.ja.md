# mouse-hide-while-typing

**Language:** [English](../../en/config/mouse-hide-while-typing.en.md) | 日本語

## 説明

入力するときにマウスを即座に非表示にします。マウスを使用する（ボタン、移動など）と、マウスが再び表示されます。プラットフォーム固有の動作は、マウスが表示される他のシナリオを決定する場合があります。たとえば macOS では、新しいウィンドウ、タブ、または分割が作成されると、マウスが再び表示されます。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/mouse-hide-while-typing.ja.md
# 入力中にマウスカーソルを非表示 (default: false)
mouse-hide-while-typing = false
```
