# font-thicken

**言語:** [English](../../en/config/font-thicken.en.md) | 日本語

## 説明

サポートされている場合、より太いストロークでフォントを描画します。これは現在のところ macOS でのみサポートされています。

## デフォルト値

```conf
# https://ghostty.org/docs/config/reference#font-thicken
# macOS でより太いストロークでフォントを描画 (default: false)
font-thicken = false
```
