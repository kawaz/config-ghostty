# resize-overlay-duration

**Language:** [English](../../en/config/resize-overlay-duration.en.md) | 日本語

## 説明

リサイズ オーバーレイが有効な場合、これはオーバーレイが画面に表示される時間を制御してから非表示になります。デフォルトは 3/4 秒または 750 ms です。

時間は、数字の後に時間単位が続く一連として指定されます。数字と単位の間に空白が許可されます。各数字と単位を合計して、合計時間を形成します。

許可される時間単位は次のとおりです：

- `y` - 365 SI 日、または 8760 時間、または 31536000 秒。うるう年またはうるう秒の調整は行われません。
- `d` - 1 SI 日、または 86400 秒。
- `h` - 1 時間、または 3600 秒。
- `m` - 1 分、または 60 秒。
- `s` - 1 秒。
- `ms` - 1 ミリ秒、または 0.001 秒。
- `us` または `µs` - 1 マイクロ秒、または 0.000001 秒。
- `ns` - 1 ナノ秒、または 0.000000001 秒。

例：
- `1h30m`
- `45s`

単位は繰り返すことができ、追加されます。つまり、`1h1h` は `2h` に相当します。これは混乱を招く可能性があるため、避ける必要があります。
今後のアップデートでは、これを禁止する可能性があります。

最大値は `584y 49w 23h 34m 33s 709ms 551µs 615ns` です。この値より大きい値は最大値に固定されます。

利用可能: 1.0.0 以降

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/resize-overlay-duration.ja.md
# リサイズオーバーレイの表示時間 (default: 750ms)
resize-overlay-duration = 750ms
```
