# 有効にする FreeType ロードフラグ。形式はコンマで区切られたフラグのリストです。
# フラグの前に `no-` を付けると無効になります。フラグを省略した場合はデフォルト値が
# 使用されるため、不要なフラグは明示的に無効にする必要があります。すべてのフラグを
# オンまたはオフにするために `true` または `false` を使用することもできます。
#
# この設定は FreeType を使用する Ghostty ビルドにのみ適用されます。
# これは通常、Linux ビルドの場合のみです。macOS は CoreText を使用しており、
# 同等の設定はありません。
#
# 利用可能なフラグ：
#
#   * `hinting` - ヒンティング (hinting) を有効または無効にします。デフォルトで有効です。
#
#   * `force-autohint` - フォント固有のヒンターではなく、常に FreeType
#     オートヒンターを使用します。デフォルトで無効です。
#
#   * `monochrome` - レンダラーに 1 ビットモノクロムレンダリングを使用するよう
#     指示します。これにより アンチエイリアシング (anti-aliasing) が無効になり、
#     ピクセルフォントを使用していない限り見栄えが良くないでしょう。
#     デフォルトで無効です。
#
#   * `autohint` - FreeType オートヒンターを有効にします。デフォルトで有効です。
#
# 例：`hinting`, `no-hinting`, `force-autohint`, `no-force-autohint`
freetype-load-flags = hinting,no-force-autohint,no-monochrome,autohint
