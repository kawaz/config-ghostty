# ファイルやフォルダをmacOSのDockのGhosttyアイコンに
# ドロップした時のウィンドウ動作を制御します。
#
# 有効な値は以下の通りです：
#
#   * `new-tab` - 現在のウィンドウに新しいタブを作成します。
#     ウィンドウが存在しない場合は新しいウィンドウを開きます。
#   * `window` - 常に新しいウィンドウを作成します。
#
# デフォルト値は `new-tab` です。
#
# この設定はmacOSでのみサポートされており、
# 他のプラットフォームでは効果がありません。
macos-dock-drop-behavior = new-tab
