# 画面のリフレッシュレートとのレンダリング同期。true の場合、ティアリング
# (画像の破裂)を最小化し、リドロー(再描画)を画面と同期させますが、入力
# レイテンシ(遅延)を引き起こす可能性があります。false の場合、リドロー頻度を
# 最大化しますが、ティアリングが発生する可能性があり、高負荷時により多く
# の CPU と電力を消費します。
#
# これは true がデフォルトです。なぜなら macOS 上の非同期レンダリングは
# カーネルパニック (macOS 14.4+) を引き起こし、DisplayLink などの
# ハードウェア経由での外部ディスプレイのパフォーマンス問題を引き起こす可能性が
# あるためです。入力レイテンシを最小化したい場合は、前述の既知リスクを承知
# の上で、これを false に設定してください。
#
# 実行時にこの値を変更すると、新しいターミナルにのみ影響します。
#
# この設定は現在のところ macOS 上でのみサポートされています。
window-vsync = true
