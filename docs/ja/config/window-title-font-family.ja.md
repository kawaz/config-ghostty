# window-title-font-family

**Language:** [English](../../en/config/window-title-font-family.en.md) | 日本語

## Description

アプリケーションのウィンドウとタブ タイトルに使用されるフォント。

この設定が設定されていない場合は、システムのデフォルト フォントが使用されます。

注: システムで利用可能なフォントを使用できます。このフォントは固定幅フォントである必要はありません。

利用可能: 1.1.0 (GTK)

## Default

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/window-title-font-family.ja.md
# ウィンドウとタブのタイトル用フォントファミリー (default: )
window-title-font-family =
```
