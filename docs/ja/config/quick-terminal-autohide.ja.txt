# フォーカスが別のウィンドウに移ったときにクイックターミナルを自動的に非表示にします。
# false に設定すると、クイックターミナルはフォーカスを失っても開いたままになります。
#
# macOS ではデフォルトで true、Linux/BSD ではデフォルトで false です。
# これは、Linux 上のグローバルショートカットはシステム設定が必要であり、
# macOS よりもアクセシビリティが大幅に低いためです。そのため、
# ユーザーがタスクを完了するまでクイックターミナルを開いたままにする方が
# より望ましいという理由によります。
# このデフォルト値は将来変わる可能性があります。
quick-terminal-autohide = true
