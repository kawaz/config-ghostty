# gtk-titlebar-hide-when-maximized

**Language:** [English](../../en/config/gtk-titlebar-hide-when-maximized.en.md) | 日本語

## 説明

これが `true` の場合、ウィンドウが最大化されるとタイトルバーが非表示になり、タイトルバーが最大化解除されると表示されます。GTK のみ。

利用可能: 1.1.0 以降

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/gtk-titlebar-hide-when-maximized.ja.md
# ウィンドウが最大化されたときにタイトルバーを非表示 (default: false)
gtk-titlebar-hide-when-maximized = false
```
