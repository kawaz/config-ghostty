# palette

**言語:** [English](../../en/config/palette.en.md) | 日本語

## 説明

多くのターミナル アプリケーションが使用する 256 色の形式のカラー パレットを設定します。この設定の構文は `N=COLOR` であり、`N` は 0 から 255（ターミナルの色テーブルの 256 色用）で、`COLOR` は `#AABBCC` や `AABBCC` などの典型的な RGB カラーコード、または名前付き X11 色です。

パレット インデックスは 10 進数、2 進数、8 進数、または 16 進数にできます。プレフィックスが使用されていない限り、10 進数と見なされます：2 進数の場合は `0b`、8 進数の場合は `0o`、16 進数の場合は `0x`。

カラー インデックスの定義と、それらが正規にマップされるものについては、[このチート シート](https://www.ditig.com/256-colors-cheat-sheet) を参照してください。

## デフォルト値

デフォルト パレットは標準的な 256 色パレットです。この設定はデフォルト パレット定義に 268 行を使用しています。完全なデフォルト パレットについては、ソース ファイルを参照してください。
