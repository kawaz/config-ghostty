# ポイント単位のフォントサイズ。この値は非整数にすることができ、
# 最も近い整数ピクセルサイズが選択される。1pt = 2px である高 DPI ディスプレイの場合、
# 半ポイントを指定することで、奇数のピクセルサイズを取得できる。
#
# 例：13.5pt @ 2px/pt = 27px
#
# 実行時にこの設定を変更すると、フォントサイズを手動で調整していない
# 既存のターミナルにのみ影響する（例：フォントサイズを増加または減少させた）。
# フォントサイズを手動で調整したターミナルは、手動で調整されたサイズを保持する。
# そうでなければ、既存のターミナルのフォントサイズはリロード時に更新される。
#
# GTK を使用した Linux では、フォントサイズはディスプレイ全体の
# スケーリング係数とテキスト固有のスケーリング係数に従ってスケーリングされる。
# これらはしばしばデスクトップ環境（例：GNOME ディスプレイスケールと
# 大きなテキスト設定）によって管理される。
font-size = 13
