# ポイント単位のフォントサイズ。この値は整数ではなく、最も近い整数の
# ピクセルサイズが選択されます。DPIが高いディスプレイで 1pt = 2px の場合、
# 小数点以下が .5 ポイントを指定することで、奇数のピクセルサイズを取得できます。
#
# 例えば、13.5pt @ 2px/pt = 27px
#
# 実行時にこの設定を変更した場合、何らかの方法でフォントサイズを手動で調整していない
# 既存のターミナルのみに影響します
# （例：フォントサイズを増加または減少させた場合）。フォントサイズを手動で
# 調整したターミナルは、手動で調整されたサイズを保持します。
# それ以外の場合は、既存のターミナルのフォントサイズは
# リロード時に更新されます。
#
# Linux（GTK）では、フォントサイズはディスプレイ全体のスケーリングと
# テキスト固有のスケーリング係数の両方に従ってスケーリングされます。これらは
# 通常、デスクトップ環境（例：GNOME のディスプレイスケールと大文字テキスト設定）
# によって管理されます。
font-size = 13
