# クイックターミナルがキーボード入力を受け取る状況を決定します。
# より詳しい説明については、対応する [Wayland ドキュメント](https://wayland.app/protocols/wlr-layer-shell-unstable-v1#zwlr_layer_surface_v1:enum:keyboard_interactivity)
# を参照してください。
#
# > [!NOTE]
# > 各オプションの正確な動作は、コンポジター (compositor) によって大きく異なる可能性があります。
# > 自分のシステムで試して、自分の好みに合ったオプションを見つけてください。
#
# 有効な値は以下の通りです：
#
#  * `none`
#
#    クイックターミナルはキーボード入力を受け取りません。
#
#  * `on-demand` (デフォルト)
#
#    クイックターミナルはフォーカスされている場合にのみキーボード入力を受け取ります。
#
#  * `exclusive`
#
#    クイックターミナルは、別のウィンドウがフォーカスされている場合でも、
#    常にキーボード入力を受け取ります。
#
# Linux Wayland でのみ効果があります。
# macOS では動作は常に `on-demand` と同等です。
#
# 利用可能バージョン：1.2.0
quick-terminal-keyboard-interactivity = on-demand
