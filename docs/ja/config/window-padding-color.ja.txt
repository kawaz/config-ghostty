# ウィンドウのパディング領域の色。有効な値：
#
# * `background` - `background`で指定された背景色。
# * `extend` - 最も近いグリッドセルの背景色を拡張します。
# * `extend-always` - "extend"と同じですが、常に拡張し、
#   以下に記載の拡張を無効にするヒューリスティックスを適用しません。
#
# "extend"の値は特定のシナリオで無効になります。プライマリ
# スクリーンアプリケーション（例：Neovimのような統合ターミナルではなく）では、
# 以下のいずれかが該当する場合、色は垂直に拡張されません：
#
# * 最も近い行にデフォルト背景色を持つセルが含まれている。
#   この場合、デフォルト背景色はパディング色として適切に見えるという考え方です。
# * 最も近い行がプロンプト行である（シェル統合が必要）。
#   プロンプトはしばしば拡張されると見栄えが悪い
#   パワーラインのグリフを含むという考え方です。
# * 最も近い行に完全にフィットするパワーライン文字が含まれている。
#   これらは拡張されると見栄えが悪いです。
window-padding-color = background
