# macos-window-shadow

**言語:** [English](../../en/config/macos-window-shadow.en.md) | 日本語

## 説明

macOS ウィンドウシャドウを有効にするかどうか。デフォルト値は true です。ウィンドウマネージャーとウィンドウの透明度の設定によっては、false の方が視覚的に魅力的に見えるかもしれません。

## デフォルト

```conf
# https://ghostty.org/docs/config/reference#macos-window-shadow
# macOS ウィンドウシャドウを有効または無効にする (default: true)
macos-window-shadow = true
```
