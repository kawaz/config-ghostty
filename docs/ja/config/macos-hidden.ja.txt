# true に設定すると、macOS のドック とアプリケーションスイッチャーのアイコンが非表示になります。
# これは主にクイックターミナルモードを使用している人を対象としています。
#
# この値を true に設定すると、キーボードレイアウトの変更は自動的に行われなくなることに注意してください。
#
# macOS アプリケーションがドックとアプリケーションスイッチャーから除外される「非表示」状態かどうかを
# 制御します。これは主にクイックターミナルモードを使用している人を対象としていますが、あらゆる
# ユースケースで使用できる一般的な設定です。
#
# 使用可能な値：
#
#   * `never` - macOS アプリケーションは決して非表示になりません。
#   * `always` - macOS アプリケーションは常に非表示になります。
#
# 注記：macOS アプリケーションが非表示の場合、キーボードレイアウトの変更は自動的に行われなくなります。
# これは macOS の制限事項です。
#
# 利用可能な版：1.2.0
macos-hidden = never
