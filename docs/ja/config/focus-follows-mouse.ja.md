# focus-follows-mouse

**Language:** [English](../../en/config/focus-follows-mouse.en.md) | 日本語

## 説明

複数の分割ペインがある場合、マウスで選択されているペインが焦点を得る場合は true です。これは現在焦点を持つウィンドウにのみ適用されます。例えば、フォーカスされていないウィンドウの分割にマウスを移動しても、その分割にフォーカスは当たらず、ウィンドウがフォアグラウンドに来ることはありません。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/focus-follows-mouse.ja.md
# 分割ビューでマウスカーソル下のペインを選択 (default: false)
focus-follows-mouse = false
```
