# 「クイック」ターミナルウィンドウの表示位置。クイックターミナル
# の詳細については、`toggle_quick_terminal` バインディング
# アクションのドキュメントを参照してください。
#
# 有効な値：
#
#   * `top` - ターミナルが画面の上部に表示されます。
#   * `bottom` - ターミナルが画面の下部に表示されます。
#   * `left` - ターミナルが画面の左側に表示されます。
#   * `right` - ターミナルが画面の右側に表示されます。
#   * `center` - ターミナルが画面の中央に表示されます。
#
# macOS では、この設定を変更するには Ghostty を完全に再起動する
# 必要があります。
#
# 注：クイックターミナルを切り替えるためのデフォルトキーバインド
# はありません。この機能を有効にするには、`toggle_quick_terminal`
# アクションをキーにバインドしてください。
quick-terminal-position = top
