# adjust-cursor-thickness

**Language:** [English](../../en/config/adjust-cursor-thickness.en.md) | 日本語

## 説明

バーカーソルと枠線付きレクトカーソルのピクセル数またはパーセンテージによる厚さ調整。

詳細は `adjust-cell-width` での調整に関する注記を参照してください。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/adjust-cursor-thickness.ja.md
# カーソルバーと矩形の厚さを調整 (default: "")
```
