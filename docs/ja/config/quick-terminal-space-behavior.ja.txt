# この設定オプションは、macOS スペース間を切り替える場合の
# クイックターミナルの動作を決定します。macOS スペースは、手動で作成された、
# または、アプリケーションがフルスクリーン モードにある場合に自動的に
# 作成される仮想デスクトップです。
#
# 有効な値は：
#
#  * `move` - 別のスペースに切り替える場合、クイックターミナルも
#    現在のスペースに移動します。
#
#  * `remain` - クイックターミナルは、元々開かれたスペースにのみ留まり、
#    別のスペースに切り替える場合は追従しません。
#
# デフォルト値は `move` です。
#
# macOS でのみ実装されています。
# Linux では、動作は常に `move` と同等です。
#
# 利用可能: 1.1.0 以降
quick-terminal-space-behavior = move
