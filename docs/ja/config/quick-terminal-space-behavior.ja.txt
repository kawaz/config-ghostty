# この設定オプションは、macOSのスペース（仮想デスクトップ）を切り替える際の
# クイックターミナルの動作を決定します。macOSのスペースは、手動で作成するか、
# アプリケーションがフルスクリーンモードの場合に自動作成される仮想デスクトップです。
#
# 有効な値は次のとおりです：
#
#  * `move` - 別のスペースに切り替えると、クイックターミナルも現在のスペースに
#    移動します。
#
#  * `remain` - クイックターミナルはもともと開いていたスペースにのみ留まり、
#    別のスペースに切り替えるときについていきません。
#
# デフォルト値は `move` です。
#
# macOSでのみ実装されています。
# Linuxではこの動作は常に `move` と同等です。
#
# 利用可能なバージョン：1.1.0
quick-terminal-space-behavior = move
