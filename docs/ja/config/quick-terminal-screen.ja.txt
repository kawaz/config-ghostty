# クイックターミナルが表示されるべきスクリーン。
#
# 有効な値は以下の通りです:
#
#  * `main` - オペレーティングシステムがメインスクリーンとして推奨するスクリーン。
#    macOSでは、現在キーボード入力を受け取っているスクリーンです。
#    このスクリーンはオペレーティングシステムによって定義され、
#    Ghosttyによって選択されるものではありません。
#
#  * `mouse` - マウスが現在ホバーしているスクリーン。
#
#  * `macos-menu-bar` - macOSのディスプレイ設定で設定されたmacOSメニューバーを
#    含むスクリーン。macOS上のすべてのスクリーンにはメニューバーが
#    ありますが、これはプライマリメニューバーを含むスクリーンです。
#
# デフォルト値は `main` です。これはオペレーティングシステムによって
# 推奨されるスクリーンだからです。
#
# macOSでのみ実装されています。
quick-terminal-screen = main
