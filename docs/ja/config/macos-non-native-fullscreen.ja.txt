# false 以外の値に設定した場合、macOS のフルスクリーンモードは
# ネイティブフルスクリーンを使用せず、アニメーションなしで新しいスペースを
# 使用してウィンドウをフルスクリーンにします。アニメーションを使用しないため、
# ネイティブフルスクリーンモードよりも高速です。
#
# 重要: このモードではタブは機能しません。ネイティブ以外のフルスクリーンは
# タイトルバーを削除し、macOS ネイティブタブはタイトルバーを必要とします。
# タブを使用する場合は、このモードを使用しないでください。
#
# タブ付きウィンドウをフルスクリーンにした場合、現在フォーカスされている
# タブがフルスクリーンになり、他のタブは背景の別のウィンドウに留まります。
# command+tilde などの通常のウィンドウ切り替えキーバインディングを使用して
# そのウィンドウに切り替えることができます。フルスクリーンを終了すると、
# ウィンドウは以前のタブ状態に戻ります。
#
# 使用可能な値は次のとおりです:
#
#   * `true` - ネイティブ以外の macOS フルスクリーンを使用し、メニューバーを隠す
#   * `false` - ネイティブ macOS フルスクリーンを使用する
#   * `visible-menu` - ネイティブ以外の macOS フルスクリーンを使用し、メニューバーを
#     表示したままにする
#   * `padded-notch` - ネイティブ以外の macOS フルスクリーンを使用し、メニューバーを
#     隠すが、該当デバイスのノッチによってウィンドウが隠れないようにする。
#     ノッチ周辺の領域は現在透明のままですが、将来的にはウィンドウの背景色で
#     埋める可能性があります。
#
# このオプションを実行時に変更できますが、次回ウィンドウがフルスクリーン
# になった時に適用されます。ウィンドウが既にフルスクリーン状態の場合、
# フルスクリーンが終了されるまで前の設定が保持されます。
macos-non-native-fullscreen = false
