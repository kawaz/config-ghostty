# unfocused-split-opacity

**言語:** [English](../../en/config/unfocused-split-opacity.en.md) | 日本語

## 説明

フォーカスされていないスプリットの不透明度レベル（透明度の反対）を設定します。フォーカスされていないスプリットはデフォルトではわずかにフェードアウトされており、どのスプリットがフォーカスされているかを簡単に確認できるようになっています。この機能を無効にするには、この値を 1 に設定します。

1 の値は完全に不透明で、0 の値は完全に透明です。「0」は有用ではない（ウィンドウが非常に奇妙に見える）ため、最小値は 0.15 です。この値はまだ奇妙に見えますが、少なくとも何が起こっているのかを見ることができます。0.15 から 1 の範囲外の値は最も近い有効な値にクランプされます。

## デフォルト値

```conf
# https://ghostty.org/docs/config/reference#unfocused-split-opacity
# フォーカスされていないスプリットペインの不透明度レベル (default: 0.7)
unfocused-split-opacity = 0.7
```
