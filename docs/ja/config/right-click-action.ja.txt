# ユーザーが端末の表面を右クリックしたときに実行するアクション。
#
# 有効な値:
#   * `context-menu` - コンテキストメニューを表示する。
#   * `paste` - クリップボードの内容をペーストする。
#   * `copy` - 選択されたテキストをクリップボードにコピーする。
#   * `copy-or-paste` - 選択がある場合、選択されたテキストをクリップボードに
#      コピーする。そうでない場合、クリップボードの内容をペーストする。
#   * `ignore` - 何もしない、右クリックを無視する。
#
# デフォルト値は `context-menu` です。
right-click-action = context-menu
