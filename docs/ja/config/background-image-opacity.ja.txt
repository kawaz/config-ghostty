# 背景画像の不透明度。
#
# これは `background-opacity` の値に対する相対値です。
#
# `1.0`（デフォルト値）の場合、背景画像は一般的な背景色の上に配置され、
# その後、結合された結果が `background-opacity` で指定された不透明度に調整されます。
#
# `1.0` より小さい値の場合、背景画像は一般的な背景色と混合された後に、
# 結合された結果が設定済みの `background-opacity` に調整されます。
#
# `1.0` より大きい値の場合、背景画像は一般的な背景色より高い不透明度を持ちます。
# 例えば、`background-opacity` が `0.5` で `background-image-opacity` が
# `1.5` に設定されている場合、背景画像の最終的な不透明度は
# `0.5 * 1.5 = 0.75` となります。
#
# 利用可能なバージョン: 1.2.0
background-image-opacity = 1
