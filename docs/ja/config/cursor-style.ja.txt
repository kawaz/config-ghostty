# カーソルのスタイル。これはデフォルトスタイルを設定します。実行中のプログラムは
# エスケープシーケンス（例：`CSI q`）を使用して、明示的なカーソルスタイルをリクエスト
# することができます。シェルの設定では、特定のカーソルスタイルをリクエストすることが
# よくあります。
#
# シェル統合は、この設定に関係なく、プロンプト時に自動的にカーソルをバー状に設定します。
# この動作を無効にするには、`shell-integration-features = no-cursor` を指定するか、
# シェル統合を完全に無効にしてください。
#
# 有効な値は以下の通りです：
#
#   * `block`
#   * `bar`
#   * `underline`
#   * `block_hollow`
cursor-style = block
