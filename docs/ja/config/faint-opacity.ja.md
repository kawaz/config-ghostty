# faint-opacity

**Language:** [English](../../en/config/faint-opacity.en.md) | 日本語

## Description

か弱いテキストの不透明度レベル (透明度の反対)。値 1 は完全に不透明で、値 0 は完全に透明です。0 より小さい値または 1 より大きい値は、最も近い有効な値にクランプされます。

Ghostty 1.2.0 以降で利用可能。

## Default

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/faint-opacity.ja.md
# か弱いテキストの不透明度レベルを制御 (default: 0.5)
faint-opacity = 0.5
```
