# GTKタイトルバーのスタイル。利用可能な値は `native` と `tabs` です。
#
# `native` タイトルバースタイルは、タイトル、いくつかのボタン、およびウィンドウコントロールを
# 含む従来のタイトルバーです。ウィンドウで複数のタブを開いている場合、
# タイトルバーの下に別のタブバーが表示されます。
#
# `tabs` タイトルバーは、タブバーと従来のタイトルバーを統合します。
# 複数のタブを使用する場合、画面の垂直スペースを節約できます。`tabs` タイトルバーの
# 制限は、タイトルによってタイトルバーをドラッグできなくなる点です
# （タイトルがタブタイトルになったため）。`tabs` タイトルバーの他の領域を使用して、
# ウィンドウをドラッグできます。
#
# デフォルトのスタイルは `native` です。
gtk-titlebar-style = native
