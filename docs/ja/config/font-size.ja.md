# font-size

**言語:** [English](../../en/config/font-size.en.md) | 日本語

## 説明

ターミナルのフォントサイズをポイント単位で設定します。この値は非整数にすることができ、最も近い整数ピクセルサイズが選択されます。1pt = 2px である高 DPI ディスプレイの場合、半ポイントを指定することで、奇数のピクセルサイズを取得できます。

例：13.5pt @ 2px/pt = 27px

実行時にこの設定を変更すると、フォントサイズを手動で調整していない既存のターミナルにのみ影響します（例：フォントサイズを増加または減少させた）。フォントサイズを手動で調整したターミナルは、手動で調整されたサイズを保持します。そうでなければ、既存のターミナルのフォントサイズはリロード時に更新されます。

GTK を使用した Linux では、フォントサイズはディスプレイ全体のスケーリング係数とテキスト固有のスケーリング係数に従ってスケーリングされます。これらはしばしばデスクトップ環境（例：GNOME ディスプレイスケールと大きなテキスト設定）によって管理されます。

## デフォルト値

```conf
# https://ghostty.org/docs/config/reference#font-size
# ターミナルのフォントサイズ（ポイント単位） (default: 13)
font-size = 13
```
