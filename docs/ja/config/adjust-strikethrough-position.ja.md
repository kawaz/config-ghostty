# adjust-strikethrough-position

**言語:** [English](../../en/config/adjust-strikethrough-position.en.md) | 日本語

## 説明

セルの上部から取り消し線の上部までの、ピクセルまたはパーセンテージの調整距離。取り消し線を下に移動させるには増加させ、上に移動させるには減少させます。

[`adjust-cell-width`](adjust-cell-width.ja.md) の調整に関するメモを参照してください。

## デフォルト

```conf
# https://ghostty.org/docs/config/reference#adjust-strikethrough-position
# セルの上部から取り消し線の位置を調整 (default: )
adjust-strikethrough-position =
```
