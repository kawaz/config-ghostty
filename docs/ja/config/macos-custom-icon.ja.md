# macos-custom-icon

**Language:** [English](../../en/config/macos-custom-icon.en.md) | 日本語

## 説明

カスタムアイコンファイルへの絶対パス。

サポートされている形式には PNG、JPEG、ICNS が含まれます。

デフォルトは `~/.config/ghostty/Ghostty.icns` です

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/macos-custom-icon.ja.md
# カスタム macOS アプリアイコンへのパス (default: ~/.config/ghostty/Ghostty.icns)
macos-custom-icon =
```
