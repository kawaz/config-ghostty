# 背景画像のフィット方法。
#
# 有効な値：
#
#  * `contain`
#
#    アスペクト比を保持して、背景画像をターミナル内に完全に収まる最大
#    サイズにスケールします。画像全体が表示されます。
#
#  * `cover`
#
#    アスペクト比を保持して、背景画像をターミナル全体をすべてカバーする
#    最小サイズにスケールします。このため、画像の1つ以上のエッジが
#    ターミナルのエッジでクリップされる可能性があります。
#
#  * `stretch`
#
#    背景画像をターミナルの全体サイズに伸ばします。アスペクト比は
#    保持されません。
#
#  * `none`
#
#    背景画像をスケールしません。
#
# デフォルト値は `contain` です。
#
# 利用可能: 1.2.0 以降
background-image-fit = contain
