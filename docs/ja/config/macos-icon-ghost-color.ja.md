# macos-icon-ghost-color

**言語:** [English](../../en/config/macos-icon-ghost-color.en.md) | 日本語

## 説明

macOS アプリケーション アイコン内のゴーストの色。

注：この設定は、`macos-icon` が `custom-style` に設定されている場合は必須です。

16 進数（`#RRGGBB` または `RRGGBB`）または X11 の名前付きカラーで指定されます。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/macos-icon-ghost-color.ja.md
# macOS アプリケーション アイコン内のゴーストの色 (default: )
macos-icon-ghost-color =
```
