# adjust-underline-position

**言語:** [English](../../en/config/adjust-underline-position.en.md) | 日本語

## 説明

セルの上部からアンダーラインの上部までの距離をピクセルまたはパーセンテージで調整。増加させてアンダーラインを下に移動、減少させて上に移動。[`adjust-cell-width`](adjust-cell-width.ja.md) の調整に関する注記を参照。

## デフォルト

```conf
# https://ghostty.org/docs/config/reference#adjust-underline-position
# アンダーラインの垂直位置を調整する (デフォルト: (なし))
adjust-underline-position =
```
