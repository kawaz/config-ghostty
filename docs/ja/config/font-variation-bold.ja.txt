# 1つまたは複数のUnicodeコードポイントを特定の名前付きフォントに
# マップしることを強制します。特殊なシンボルをサポートしたい場合や、
# 特定のフォントでより良く描画される特定のグリフを使用したい場合に
# 便利です。
#
# 構文は `codepoint=fontname` です。ここで `codepoint` は単一の
# コードポイントまたはコードポイント範囲です。コードポイントは
# `U+ABCD` のような完全なUnicodeの16進値で指定する必要があります。
# コードポイント範囲は `U+ABCD-U+DEFG` として指定されます。複数の範囲を
# カンマで区切って同じフォントに指定できます。例えば
# `U+ABCD-U+DEFG,U+1234-U+5678=fontname` のように指定できます。
# フォント名は `font-family` に使用する値と同じです。
#
# この設定は複数回指定して、複数のコードポイント
# マッピングを定義できます。
#
# この設定を実行時に変更した場合、新しいターミナルにのみ
# 反映されます。すなわち新しいウィンドウ、タブなどが対象です。
