# ウィンドウに使用するテーマ。有効な値：
#
#   * `auto` - 設定されたターミナル背景色に基づいてテーマを決定します。「theme」
#      設定に別々のライトテーマとダークテーマがある場合、これは効果がありません。
#      その場合、「auto」の動作は「system」と同等です。
#   * `system` - システムテーマを使用します。
#   * `light` - システムテーマに関わらずライトテーマを使用します。
#   * `dark` - システムテーマに関わらずダークテーマを使用します。
#   * `ghostty` - Ghostty設定で指定された背景色と前景色を使用します。
#     これはLinuxビルドでのみサポートされています。
#
# macOS上で、`macos-titlebar-style`が「tabs」の場合、ウィンドウテーマは
# ターミナル背景色の明度に基づいて自動的に設定されます。
# これはターミナルウィンドウにのみ適用されます。このこの設定は引き続き
# Ghostty内の非ターミナルウィンドウに適用されます。
#
# 現在のところ、macOSとLinuxでのみサポートされています。
window-theme = auto
