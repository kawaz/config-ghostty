# 多くのターミナル アプリケーションが使用する 256 色の形式のカラー パレット。
# この設定の構文は `N=COLOR` であり、`N` は 0 から 255（ターミナルの
# 色テーブルの 256 色用）で、`COLOR` は `#AABBCC` や `AABBCC` などの
# 典型的な RGB カラーコード、または名前付き X11 色である。
#
# パレット インデックスは 10 進数、2 進数、8 進数、または 16 進数にできる。
# プレフィックスが使用されていない限り、10 進数と見なされる：
# 2 進数の場合は `0b`、8 進数の場合は `0o`、16 進数の場合は `0x`。
#
# カラー インデックスの定義と、それらが正規にマップされるもの、
# については、[このチート シート](https://www.ditig.com/256-colors-cheat-sheet) を参照。
