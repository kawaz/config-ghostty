# adjust-font-baseline

**Language:** [English](../../en/config/adjust-font-baseline.en.md) | 日本語

## 説明

セルの下部からテキストベースラインまでのピクセル数またはパーセンテージによる距離調整。増加させるとベースラインが上に移動し、減少させると下に移動します。

詳細は `adjust-cell-width` での調整に関する注記を参照してください。

## デフォルト

```conf
# https://github.com/kawaz/config-ghostty/blob/main/docs/ja/config/adjust-font-baseline.ja.md
# セルの下部からテキストベースラインまでの距離を調整 (default: "")
adjust-font-baseline =
```
