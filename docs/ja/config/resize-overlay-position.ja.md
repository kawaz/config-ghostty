# resize-overlay-position

**言語:** [English](../../en/config/resize-overlay-position.en.md) | 日本語

## 説明

リサイズオーバーレイが有効になっている場合、オーバーレイの位置を制御。可能なオプション：

- `center`
- `top-left`
- `top-center`
- `top-right`
- `bottom-left`
- `bottom-center`
- `bottom-right`

デフォルトは `center`。

## デフォルト

```conf
# https://ghostty.org/docs/config/reference#resize-overlay-position
# ウィンドウリサイズオーバーレイの位置 (デフォルト: center)
resize-overlay-position = center
```
