# copy_url_to_clipboard

**Language:** English | [日本語](../../ja/actions/copy_url_to_clipboard.ja.md)

## Description

If there is a URL under the cursor, copy it to the default clipboard.

## Usage

```conf
keybind = ctrl+shift+u=copy_url_to_clipboard
```
