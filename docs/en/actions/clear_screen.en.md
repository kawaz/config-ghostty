# clear_screen

**Language:** English | [日本語](../../ja/actions/clear_screen.ja.md)

## Description

Clear the screen and all scrollback.

## Usage

This action is invoked without arguments:

```conf
# https://ghostty.org/docs/config/reference#clear_screen
# Clear the screen and all scrollback
unbind ctrl+l
keybind ctrl+shift+k = clear_screen
```
