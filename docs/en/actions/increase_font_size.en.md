# increase_font_size

**Language:** English | [日本語](../../ja/actions/increase_font_size.ja.md)

## Description

Increase the font size by the specified amount in points (pt).

For example, `increase_font_size:1.5` will increase the font size by 1.5 points.

## Example

```conf
keybind = super+plus:increase_font_size:1
keybind = super+equal:increase_font_size:2
```
