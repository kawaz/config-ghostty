# toggle_maximize

**Language:** English | [日本語](../../ja/actions/toggle_maximize.ja.md)

## Description

Maximize or unmaximize the current window.

This has no effect on macOS as it does not have the concept of maximized windows.

## Usage

```conf
toggle_maximize
```
