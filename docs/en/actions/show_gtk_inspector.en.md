# show_gtk_inspector

**Language:** English | [日本語](../../ja/actions/show_gtk_inspector.ja.md)

## Description

Show the GTK inspector.

Has no effect on macOS.

## Example

```conf
# https://ghostty.org/docs/config/reference#show_gtk_inspector
# Show the GTK inspector
keybind = super+i=show_gtk_inspector
```
