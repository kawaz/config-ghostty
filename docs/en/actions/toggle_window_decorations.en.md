# toggle_window_decorations

**Language:** English | [日本語](../../ja/actions/toggle_window_decorations.ja.md)

## Description

Toggle window decorations (titlebar, buttons, etc.) for the current window.

Only implemented on Linux.

## Example

```conf
keybind = alt+f11=toggle_window_decorations
```
