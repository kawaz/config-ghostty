# new_window

**Language:** English | [日本語](../../ja/actions/new_window.ja.md)

## Description

Open a new window.

If the application isn't currently focused, this will bring it to the front.

## Example

```conf
keybind = cmd+n=new_window
keybind = ctrl+alt+n=new_window
```
