# copy_to_clipboard

**Language:** English | [日本語](../../ja/actions/copy_to_clipboard.ja.md)

## Description

Copy the selected text to the clipboard.

## Usage

This action is invoked without arguments:

```conf
# https://ghostty.org/docs/config/reference#copy_to_clipboard
# Copy selected text to clipboard
keybind ctrl+shift+c = copy_to_clipboard
```
