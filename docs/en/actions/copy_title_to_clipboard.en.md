# copy_title_to_clipboard

**Language:** English | [日本語](../../ja/actions/copy_title_to_clipboard.ja.md)

## Description

Copy the terminal title to the clipboard. If the terminal title is not set or is empty this has no effect.

## Example

```conf
# https://ghostty.org/docs/config/reference#copy_title_to_clipboard
# Copy the terminal title to the clipboard
keybind = super+c=copy_title_to_clipboard
```
