# inspector

**Language:** English | [日本語](../../ja/actions/inspector.ja.md)

## Description

Control the visibility of the terminal inspector.

Valid arguments: `toggle`, `show`, `hide`.

## Usage

This action requires a visibility control argument:

```conf
# https://ghostty.org/docs/config/reference#inspector
# Control terminal inspector
keybind f12 = inspector:toggle
```
