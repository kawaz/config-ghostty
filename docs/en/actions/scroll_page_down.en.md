# scroll_page_down

**Language:** English | [日本語](../../ja/actions/scroll_page_down.ja.md)

## Description

Scroll the screen down by one page.

## Usage

This action is invoked without arguments:

```conf
# https://ghostty.org/docs/config/reference#scroll_page_down
# Scroll down by one page
keybind page_down = scroll_page_down
```
