# set_font_size

**Language:** English | [日本語](../../ja/actions/set_font_size.ja.md)

## Description

Set the font size to the specified size in points (pt).

For example, `set_font_size:14.5` will set the font size to 14.5 points.

## Example

```conf
keybind = super+1:set_font_size:12
keybind = super+2:set_font_size:14
keybind = super+3:set_font_size:16
```
