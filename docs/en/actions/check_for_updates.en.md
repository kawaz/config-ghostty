# check_for_updates

**Language:** English | [日本語](../../ja/actions/check_for_updates.ja.md)

## Description

Check for updates.

Only implemented on macOS.

## Usage

This action is invoked without arguments:

```conf
# https://ghostty.org/docs/config/reference#check_for_updates
# Check for available updates
keybind alt+shift+u = check_for_updates
```

## Platform

macOS only
