# foreground

**Language:** English | [日本語](../../ja/config/foreground.ja.md)

## Description

Foreground color for the window.
Specified as either hex (`#RRGGBB` or `RRGGBB`) or a named X11 color.

## Default

```conf
# https://ghostty.org/docs/config/reference#foreground
# Foreground text color (default: "#ffffff")
foreground = #ffffff
```
