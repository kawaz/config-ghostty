# macos-custom-icon

**Language:** English | [日本語](../../ja/config/macos-custom-icon.ja.md)

## Description

The absolute path to the custom icon file.

Supported formats include PNG, JPEG, and ICNS.

Defaults to `~/.config/ghostty/Ghostty.icns`

## Default

```conf
# https://ghostty.org/docs/config/reference#macos-custom-icon
# Path to custom macOS app icon (default: ~/.config/ghostty/Ghostty.icns)
macos-custom-icon =
```
