# gtk-toolbar-style

**Language:** English | [日本語](../../ja/config/gtk-toolbar-style.ja.md)

## Description

Determines the appearance of the top and bottom bars tab bar.

Valid values are:

* `flat` - Top and bottom bars are flat with the terminal window.
* `raised` - Top and bottom bars cast a shadow on the terminal area.
* `raised-border` - Similar to `raised` but the shadow is replaced with a more subtle border.

## Default

```conf
# https://ghostty.org/docs/config/reference#gtk-toolbar-style
# Appearance of GTK toolbar and tab bars (default: raised)
gtk-toolbar-style = raised
```
