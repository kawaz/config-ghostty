# adjust-cursor-height

**Language:** English | [日本語](../../ja/config/adjust-cursor-height.ja.md)

## Description

Height in pixels or percentage adjustment of the cursor. Currently applies to all cursor types: bar, rect, and outlined rect. See the notes about adjustments in `adjust-cell-width`.

## Default

```conf
# https://ghostty.org/docs/config/reference#adjust-cursor-height
# Cursor height adjustment (default: unset)
adjust-cursor-height =
```
