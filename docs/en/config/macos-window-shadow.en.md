# macos-window-shadow

**Language:** English | [日本語](../../ja/config/macos-window-shadow.ja.md)

## Description

Whether to enable the macOS window shadow. The default value is true. With some window managers and window transparency settings, you may find false more visually appealing.

## Default

```conf
# https://ghostty.org/docs/config/reference#macos-window-shadow
# Enable or disable macOS window shadow (default: true)
macos-window-shadow = true
```
