# background

**Language:** English | [日本語](../../ja/config/background.ja.md)

## Description

Background color for the window. Specified as either hex (`#RRGGBB` or `RRGGBB`) or a named X11 color.

## Default

```conf
# https://ghostty.org/docs/config/reference#background
# Background color (default: #282c34)
background = #282c34
```
