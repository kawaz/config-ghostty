# resize-overlay-position

**Language:** English | [日本語](../../ja/config/resize-overlay-position.ja.md)

## Description

If resize overlays are enabled, this controls the position of the overlay. The possible options are:

- `center`
- `top-left`
- `top-center`
- `top-right`
- `bottom-left`
- `bottom-center`
- `bottom-right`

The default is `center`.

## Default

```conf
# https://ghostty.org/docs/config/reference#resize-overlay-position
# Position of the window resize overlay (default: center)
resize-overlay-position = center
```
