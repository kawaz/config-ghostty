# split-divider-color

**Language:** English | [日本語](../../ja/config/split-divider-color.ja.md)

## Description

The color of the split divider. If this is not set, a default will be chosen. Specified as either hex (`#RRGGBB` or `RRGGBB`) or a named X11 color.

Available since: 1.1.0

## Default

```conf
# https://ghostty.org/docs/config/reference#split-divider-color
# The color of the split divider (default: (platform dependent))
split-divider-color =
```
