# gtk-opengl-debug

**Language:** English | [日本語](../../ja/config/gtk-opengl-debug.ja.md)

## Description

Enable or disable GTK's OpenGL debugging logs. The default is `true` for debug builds, `false` for all others.

Available since: 1.1.0

## Default

```conf
# https://ghostty.org/docs/config/reference#gtk-opengl-debug
# Enable or disable GTK OpenGL debugging logs (default: false)
gtk-opengl-debug = false
```
