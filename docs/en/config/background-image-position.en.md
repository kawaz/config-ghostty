# background-image-position

**Language:** English | [日本語](../../ja/config/background-image-position.ja.md)

## Description

Background image position.

Valid values are:
- `top-left`
- `top-center`
- `top-right`
- `center-left`
- `center`
- `center-right`
- `bottom-left`
- `bottom-center`
- `bottom-right`

The default value is `center`.

Available since: 1.2.0

## Default

```conf
# https://ghostty.org/docs/config/reference#background-image-position
# Position of the background image (default: center)
background-image-position = center
```
