# desktop-notifications

**Language:** English | [日本語](../../ja/config/desktop-notifications.ja.md)

## Description

If `true` (default), applications running in the terminal can show desktop notifications using certain escape sequences such as OSC 9 or OSC 777.

## Default

```conf
# https://ghostty.org/docs/config/reference#desktop-notifications
# Allow terminal applications to show desktop notifications (default: true)
desktop-notifications = true
```
