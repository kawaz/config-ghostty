# macos-icon-ghost-color

**Language:** English | [日本語](../../ja/config/macos-icon-ghost-color.ja.md)

## Description

The color of the ghost in the macOS app icon.

Note: This configuration is required when `macos-icon` is set to `custom-style`.

Specified as either hex (`#RRGGBB` or `RRGGBB`) or a named X11 color.

## Default

```conf
# https://ghostty.org/docs/config/reference#macos-icon-ghost-color
# Color of ghost in macOS app icon (default: )
macos-icon-ghost-color =
```
