# window-title-font-family

**Language:** English | [日本語](../../ja/config/window-title-font-family.ja.md)

## Description

The font that will be used for the application's window and tab titles.

If this setting is left unset, the system default font will be used.

Note: any font available on the system may be used, this font is not required to be a fixed-width font.

Available since: 1.1.0 (on GTK)

## Default

```conf
# https://ghostty.org/docs/config/reference#window-title-font-family
# Font family for window and tab titles (default: )
window-title-font-family =
```
