# window-new-tab-position

**Language:** English | [日本語](../../ja/config/window-new-tab-position.ja.md)

## Description

The position where new tabs are created. Valid values:

* `current` - Insert the new tab after the currently focused tab, or at the end if there are no focused tabs.
* `end` - Insert the new tab at the end of the tab list.

## Default

```conf
# https://ghostty.org/docs/config/reference#window-new-tab-position
# Position for newly created tabs (default: current)
window-new-tab-position = current
```
