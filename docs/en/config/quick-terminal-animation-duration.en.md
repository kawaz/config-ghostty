# quick-terminal-animation-duration

**Language:** English | [日本語](../../ja/config/quick-terminal-animation-duration.ja.md)

## Description

Duration (in seconds) of the quick terminal enter and exit animation. Set it to 0 to disable animation completely. This can be changed at runtime.

Only implemented on macOS.

## Default

```conf
# https://ghostty.org/docs/config/reference#quick-terminal-animation-duration
# Duration of quick terminal animation (default: 0.2)
quick-terminal-animation-duration = 0.2
```
