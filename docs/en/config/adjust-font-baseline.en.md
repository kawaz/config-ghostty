# adjust-font-baseline

**Language:** English | [日本語](../../ja/config/adjust-font-baseline.ja.md)

## Description

Distance in pixels or percentage adjustment from the bottom of the cell to the text baseline. Increase to move baseline UP, decrease to move baseline DOWN.

See the notes about adjustments in `adjust-cell-width`.

## Default

```conf
# https://ghostty.org/docs/config/reference#adjust-font-baseline
# Adjust text baseline distance from cell bottom (default: "")
adjust-font-baseline =
```
