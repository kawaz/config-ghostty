# adjust-strikethrough-position

**Language:** English | [日本語](../../ja/config/adjust-strikethrough-position.ja.md)

## Description

Distance in pixels or percentage adjustment from the top of the cell to the top of the strikethrough. Increase to move strikethrough DOWN, decrease to move strikethrough UP.

See the notes about adjustments in [`adjust-cell-width`](adjust-cell-width.en.md).

## Default

```conf
# https://ghostty.org/docs/config/reference#adjust-strikethrough-position
# Adjust strikethrough position from the top of the cell (default: )
adjust-strikethrough-position =
```
