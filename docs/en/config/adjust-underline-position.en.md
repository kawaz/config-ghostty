# adjust-underline-position

**Language:** English | [日本語](../../ja/config/adjust-underline-position.ja.md)

## Description

Distance in pixels or percentage adjustment from the top of the cell to the top of the underline. Increase to move underline DOWN, decrease to move underline UP. See the notes about adjustments in [`adjust-cell-width`](adjust-cell-width.en.md).

## Default

```conf
# https://ghostty.org/docs/config/reference#adjust-underline-position
# Adjust the vertical position of underline (default: (none))
adjust-underline-position =
```
