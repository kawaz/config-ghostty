# font-thicken

**Language:** English | [日本語](../../ja/config/font-thicken.ja.md)

## Description

Draw fonts with a thicker stroke, if supported. This is currently only supported on macOS.

## Default

```conf
# https://ghostty.org/docs/config/reference#font-thicken
# Draw fonts with a thicker stroke on macOS (default: false)
font-thicken = false
```
