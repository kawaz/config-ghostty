# gtk-titlebar-hide-when-maximized

**Language:** English | [日本語](../../ja/config/gtk-titlebar-hide-when-maximized.ja.md)

## Description

If this is `true`, the titlebar will be hidden when the window is maximized, and shown when the titlebar is unmaximized. GTK only.

Available since: 1.1.0

## Default

```conf
# https://ghostty.org/docs/config/reference#gtk-titlebar-hide-when-maximized
# Hide titlebar when window is maximized (default: false)
gtk-titlebar-hide-when-maximized = false
```
